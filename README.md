#  Multi-Threaded Port Scanner

A professional Python port scanner with multi-threading, logging, and exception handling.

## Features
- ⚡ **Multi-threaded** — scans 100 ports at once!
- 📝 **Auto-logs** results to `scan_results.txt`
- 🛡️ **Safe** — handles errors gracefully
- 🎯 **Easy to use** — just type and go!

##  How to Run
```bash
python port_scanner.py
